# Shard

Enhanced tab and editor splitter management for IntelliJ IDEA.

## Features

- **Reorder Tabs**: Quickly reorganize tabs within your editor
- **Move Tabs Between Splitters**: Transfer tabs between neighboring split editor panes
- **Smart Neighbor Selection**: Configurable strategy for selecting target splitters when moving tabs

Shard streamlines your workflow by providing intuitive actions for managing editor tabs and split views, helping you
maintain an organized workspace as you navigate complex projects.
